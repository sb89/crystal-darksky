# crystal-darksky

Dark Sky API wrapper in Crystal.

Documentation: https://darksky.net/dev/docs

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  darksky:
    github: sb89/crystal-darksky
```

